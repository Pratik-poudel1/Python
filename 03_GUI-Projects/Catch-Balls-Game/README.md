# Catch the Falling Balls Game

## Introduction

This is a simple game made using Python and the pygame library. The goal of this project is to learn the basic idea of game development using pygame. In this game, balls fall from the top of the screen and the player needs to catch them using a paddle at the bottom.

## How the Game Works

The game window opens and balls start falling from the top of the screen. The player controls a paddle placed at the bottom. The paddle can move left and right using the keyboard arrow keys.

If the paddle catches the ball, the score increases. If the ball is missed, it resets and starts falling again from the top.

## What I Learned

While making this project, I learned the basic structure of a pygame program. This includes creating the game window, running a game loop, drawing shapes on the screen, handling keyboard input, checking collision between objects, and displaying the score.

## Requirements

Python
pygame

## Installation

Install pygame using the following command in the terminal.

```
pip install pygame
```

## Running the Game

After installing pygame, run the program using this command.

```
python catch_balls.py
```

## Purpose

This project was created as a beginner practice project to understand the basics of pygame and simple game logic.
