# Greeting GUI with CustomTkinter

## Project Overview

This is a simple graphical user interface program made with Python and CustomTkinter.  
The program opens a window where a user can type their name and press a button to receive a greeting message.

The main goal of this project is to learn the basic ideas of GUI programming in Python.

## What the Program Does

When the program runs, a window appears.

The window asks the user to enter a name.  
The user types their name in the input box and presses the **Greet** button.

After pressing the button, the program shows a greeting message.  
It also displays a short message that the program is using a CustomTkinter GUI and Python 3.13.

## Technologies Used

Python  
CustomTkinter

## Concepts Learned

Creating a GUI window  
Using labels, buttons, and input boxes  
Getting text from an input field  
Updating text in a label  
Using a button command to run a function

## How to Run the Program

1. Install Python on your computer.

2. Install CustomTkinter by running this command in the terminal:

```
pip install customtkinter
```

3. Save the program file as:

```
Greeting_GUI.py
```

4. Run the program with this command:

```
python Greeting_GUI.py
```

## Example Output

A window will open with a text input and a button.  
After entering a name and pressing the button, the window will display a greeting message.
