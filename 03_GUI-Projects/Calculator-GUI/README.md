# GUI Calculator

Hi there! 

This is a simple **GUI calculator** I built using **Python** and **CustomTkinter**. I made it as a small project to practice GUI programming and basic Python logic.  

## Features

- Basic arithmetic operations: `+`, `-`, `*`, `/`  
- Decimal point support  
- Clear button to reset the input  
- Simple and easy-to-use interface  

## How to Use

1. Run `Calculator.py` using Python 3:

   ```bash
   python Calculator.py
