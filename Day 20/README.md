# Automated File Organizer

A Python automation script that scans a folder and automatically organizes files into categorized subfolders such as Images, Documents, Videos, Music, Archives, and Others.

This project helps keep directories clean and structured by sorting files based on their extensions.

---

## Objective

The goal of this project is to practice **Python automation** by creating a script that can:

1. Scan a folder
2. Detect file types
3. Create folders automatically
4. Move files into appropriate categories

---

## Technologies Used

1. Python
2. pathlib
3. shutil

---

## Features

A. Automatically scans a selected folder
<br>
B. Detects file types using extensions
<br>
C. Creates folders if they don't exist
<br>
D. Organizes files into categories:
<br>
  1) Images
  2) Documents
  3) Videos
  4) Music
  5) Archives
  6) Others
<br>
E. Simple and beginner-friendly code
