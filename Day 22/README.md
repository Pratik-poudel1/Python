# Simple To Do List (CustomTkinter)

This is a very simple desktop To Do List made with Python and CustomTkinter.

The app lets you add tasks, mark them as done, and delete them. It runs in a small window and is easy to use.

## Features

- Add a new task  
- Show tasks in a scrollable list  
- Mark tasks as done using a checkbox  
- Delete tasks from the list  

## Requirements

- Python 3  
- CustomTkinter  

Install CustomTkinter using pip:

```bash
pip install customtkinter
