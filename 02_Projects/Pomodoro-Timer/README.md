# Pomodoro Timer

This is a simple Pomodoro Timer made with Python and CustomTkinter.

The program shows a countdown timer for 25 minutes. You can start the timer and reset it. A progress bar shows how much time has passed. When the timer finishes, the program plays a beep sound.

This project is made for learning GUI programming in Python. The code is simple so beginners can understand how a timer works in a desktop application.

## What this program does

It shows a 25 minute countdown timer.

It has a start button to begin the timer.

It has a reset button to return the timer to 25 minutes.

It shows a progress bar that fills as time passes.

It plays a sound when the timer finishes.

## Tools used

Python

CustomTkinter

winsound

## How to run the program

Install Python on your computer.

Install CustomTkinter.

Open the file pomodoro.py and run it with Python.

## Why this project exists

This project was made to practice building simple graphical programs in Python. It also helps understand how timers and event loops work in GUI applications.
